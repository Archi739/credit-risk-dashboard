# Credit Risk Scoring Dashboard

![Dashboard](app.png)

---

## Overview

This project predicts the **credit risk of loan applicants** using Machine Learning models like Logistic Regression and XGBoost.
It provides an interactive dashboard to evaluate whether a customer is **likely to default or not**.

---

## Features

* Predict loan default risk
* Interactive dashboard using Streamlit
* Real-time prediction

---

## Models Used

* Logistic Regression
* XGBoost

---

## Dataset

* Home Credit Default Risk Dataset (Kaggle)
* Dataset not included due to size limitations

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Run the App

```bash
streamlit run dashboard.py
```

---

## Project Structure

* dashboard.py → Streamlit app
* credit_risk_xgb.pkl → trained model
* scaler.pkl → preprocessing
* feature_order.pkl → feature alignment
